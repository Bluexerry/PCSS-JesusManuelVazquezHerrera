Proyectos

Proyecto 1: Plataforma de Gestión de Inventarios

Desarrollé una aplicación web completa para la gestión de inventarios utilizando React para el frontend y Node.js con Express para el backend. Implementé bases de datos MongoDB para el almacenamiento eficiente de datos y Docker para el despliegue. La plataforma permite a los usuarios realizar un seguimiento en tiempo real de los niveles de inventario, generar informes personalizados y automatizar procesos de reabastecimiento.

Proyecto 2: Sistema de Automatización para Hogares Inteligentes

Diseñé e implementé un sistema de automatización del hogar que integra dispositivos IoT para controlar iluminación, climatización y seguridad. Utilicé Python y Raspberry Pi para el desarrollo de scripts de automatización y Docker para la contenedorización de aplicaciones. El sistema incluye una interfaz de usuario intuitiva desarrollada en Angular, permitiendo a los usuarios gestionar sus hogares de manera eficiente y segura desde cualquier lugar.
