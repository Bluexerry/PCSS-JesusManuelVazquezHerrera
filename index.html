Proyectos

Framework de automatizacion de pruebas con Selenium

Desarrollé un framework de automatización de pruebas en Java utilizando Selenium WebDriver y TestNG, gestionado con Maven. Orientado al ámbito educativo, el proyecto facilita la automatización de pruebas de aplicaciones web, la gestión eficiente de casos de prueba y la generación de reportes detallados con Allure. Este framework permite interactuar con elementos web, estructurar y ejecutar pruebas de manera organizada, y enseñar conceptos fundamentales de la automatización de pruebas.

Panel avanzado de gestion de miembros

El proyecto Guild Management es una aplicación web desarrollada con React para gestionar miembros de una guild y administrar grupos para actividades. Ofrece funcionalidades como añadir, editar y listar miembros, crear y gestionar partidos, y un sistema de notificaciones para informar sobre el estado de las operaciones. Utiliza React Router para la navegación, CSS modular para los estilos y Jest junto con React Testing Library para las pruebas. La estructura del proyecto está organizada de manera modular, facilitando la escalabilidad y el mantenimiento.

API diseñada sobre .NET8 en C#

Proyecto API RESTful desarrollada en C# sobre .NET 8, diseñada para consumir la API externa de The Simpsons Quote. Su objetivo es ofrecer tres rutas principales para obtener frases de personajes de Los Simpsons, ya sea de forma aleatoria, filtradas por personaje o en una cantidad específica. Todas las respuestas se devuelven en formato JSON.
